React

    1. React (also known as React.js or ReactJS) is a free and open-source front-end JavaScript library for building user interfaces based on UI component.
    2. React was created by Jordan Walke, a software engineer at Facebook
    3. React Initial Public Released in 29 May 2019
    4. Uses virtual DOM which is a JavaScript object. This will improve apps performance, since JavaScript virtual DOM is faster than the regular DOM.
    5. Component and data patterns improve readability, which helps to maintain larger apps.
