java

  1. Java is a multi-platform, object-oriented, and network-centric language. It is among the most used programming language. Java is also used as a computing platform.
  2. Java was created at Sun Microsystems, by James Gosling and his team, Work on the language began in 1991 and released in 1995
  3. java is used for:
    • Desktop GUI Applications.
    • Web Applications.
    • Mobile Applications.
    • Web Servers & Applications Servers.
    • Embedded Systems.
    • Scientific Applications.
