Web Technologies

HTML

HTML stands for hyper text markup language

It is a markup language which is used to structured the content of a webpage

The first version of HTML was written by Tim Berners-Lee in 1993

There are so many types of elements like Image, Header, Division

CSS

CSS stands for cascade style sheet

CSS is a style sheet language responsible for the presentation of documents written in a markup language

CSS invented by World Wide Web Consortium (W3C) in 1996

javaScript

JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images.

javaScript is created by Brendan Eich in 1995

It is a a lightweight, interpreted, object-oriented language

javaScript used for creating webpage and mobile apps

React JS

React (also known as React.js or ReactJS) is a free and open-source front-end JavaScript library for building user interfaces based on UI component.

React was created by Jordan Walke, a software engineer at Facebook

React Initial Public Released in 29 May 2019

Uses virtual DOM which is a JavaScript object. This will improve apps performance, since JavaScript virtual DOM is faster than the regular DOM.

Component and data patterns improve readability, which helps to maintain larger apps.
